Sobre mi.

Me llamo Ana Clara Culeddu, tengo 19 años y soy técnica en computación a nivel secundario, además de estudiante de ingeniería de sistemas de información
No cuento con experiencias de puestos laborales. Sin embargo, en un futuro me gustaría empezar como desarrolladora web. El desarrollo web es la rama que más me apasiona.

Conocimientos.

Mis conocimientos se basan especialmente en el Front End: Manejo HTML, CSS (y el framework Bootstrap), Javascript y lo básico de Git.
Fuera de desarrollo web, tengo conocimientos sobre Python (y su librería de interfaz gráfica tkinter), SQLite y MYsql en cuanto a bases de datos y lo básico de C++ y Java.

Proyectos.

Dentro de mi Github (link en el final del portafolio), podrán encontrar ejercicios y proyectos con los que aprendí
Destaco algunos proyectos:



    · TuHorario: realizado para la materia "Proyecto Informático I", es un planner semanal programado con Python y con uso de la base de datos SQLite. Permite al usuario organizar sus horarios, con una función extra para guardar links y acceder a ellos con facilidad.


    · Random Webpages: Esta página en inglés está programada para llevarte a otro sitio web según el botón que eligas. Se encuentran los sitios divertidos, para estudiar, para ver películas y para jugar videojuegos.


    · Bibliolibro: Este trabajo fue realizado para el curso de desarrollo web 1. Si bien es un simulador de blog sencillo hecho principalmente con HTML Y CSS, tiene la característica de utilizar en su página principal elementos con Bootstrap. Le agregué un poco de código de Javascript.


    · Ann Calculator: Es una calculadora con funciones avanzadas, además de contar con animación al tener el modo oscuro.
